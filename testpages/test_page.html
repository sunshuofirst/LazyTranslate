LazyTranslate Test Page

This page is designed to test the LazyTranslate Chrome extension

Testing Instructions:

  1. Install the LazyTranslate extension in Chrome
  2. Right-click on this page and select "LazyTranslate - 翻译此页面" to translate the entire page
  3. Select any text and right-click to choose "LazyTranslate - 翻译此区域" to translate selected text
  4. Right-click and select "LazyTranslate - 显示网页原文" to restore the original page content
  5. Click the extension icon to open the popup and manage custom words
